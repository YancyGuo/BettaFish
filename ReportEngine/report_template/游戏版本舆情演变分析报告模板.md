### 游戏版本舆情演变分析报告模板

* **1.0 版本舆情总体概览**

  * 1.1 版本整体口碑概述（正向 / 中性 / 负向大致态势与核心印象）
  * 1.2 版本舆情演变主线（从爆料 / 前瞻到上线与中后期的情绪走向）
  * 1.3 本期关键发现与核心结论（对产品与运营可直接消费的几条浓缩判断）

* **2.0 版本背景与数据边界**

  * 2.1 版本基本信息（游戏名称、版本号、版本副标题、核心更新要点）
  * 2.2 分析时间范围与阶段划分说明（阶段 A / B / C 的时间区间与含义）
  * 2.3 数据来源与样本结构概览（主要平台、样本规模与明显的平台倾斜情况）

* **3.0 整体声量与情绪趋势**

  * 3.1 声量走势与关键峰值（对应爆料、前瞻、上线、活动开放、公告等节点）
  * 3.2 情绪分布与阶段性变化（基于样本情感分析的正向 / 中性 / 负向相对占比及主要趋势，可配合饼图 / 条形图）
  * 3.3 渠道差异概览（不同平台在声量与情绪上的整体差异与风格特征）

* **4.0 分阶段舆情演变分析（阶段 A / B / C）**

  * 4.1 阶段 A：版本上线前阶段（预期构建阶段）

    * 4.1.1 该阶段声量与情绪概况（爆料 / 前瞻期间的热度与整体情绪基调）
    * 4.1.2 主要讨论议题与高频观点（围绕爆料 / 前瞻 / 测试服的核心话题与典型正负观点）
    * 4.1.3 重要舆情节点、代表性声音与关键放大者（关键事件、KOL / UP 主 / 主播等放大器的作用）
  * 4.2 阶段 B：版本正式上线及前中期（体验落地阶段）

    * 4.2.1 上线及前中期声量与情绪变化概况（上线首周至前两三周的声量与情绪走势）
    * 4.2.2 玩家实际体验相关的核心议题（角色 / 数值、剧情、玩法与活动、商业化体验、技术表现等）
    * 4.2.3 关键事件、集中反馈与舆论放大者（版本上线后典型事件的舆论反应与放大者的引导作用）
  * 4.3 阶段 C：版本中后期与尾声阶段（口碑沉淀阶段）

    * 4.3.1 中后期声量与情绪整体走向（长尾期声量变化与主导情绪）
    * 4.3.2 长尾议题与持续争议点（中后期仍被反复提及的长线玩法、环境平衡、养成压力等话题）
    * 4.3.3 阶段性总结与“最终印象”形成过程（含放大者）（玩家如何在中后期形成对本版本的最终评价与定调内容）

* **5.0 核心议题与体验维度综述**

  * 5.1 角色、职业与数值平衡相关议题概览（强度评价、环境与阵容生态、典型爽点与毒点）
  * 5.2 剧情与世界观相关议题概览（故事节奏、文本质量、演出反馈与世界观推进）
  * 5.3 活动与玩法相关议题概览（新玩法 / 常驻模式 / 活动在可玩性、深度、肝度与奖励感受上的综合评价）
  * 5.4 商业化与氪金体验相关议题概览（卡池 / 抽卡、礼包 / 通行证、付费体验与福利感知）
  * 5.5 技术、服务与运营沟通体验相关议题概览（性能、BUG、服务器、外挂与公告 / 补偿沟通等）

* **6.0 风险提示与机会点洞察**` 

  * 6.1 当前版本暴露出的主要风险点（可能在后续版本或社区持续发酵的问题）
  * 6.2 可以放大的正向机会点（玩家明显认可、可在后续版本延续或强化的设计与改动方向）
  * 6.3 面向后续版本与运营的行动建议要点（从舆情视角出发的优先级排序与关键建议）
